平台基础结构
==========

前台应用为基于 Angular.js 构建的单页面应用，应用的页面路由是由前端负责。

后端服务器为无 session 的 API Server，负责对前端发送的请求进行处理和响应。

前后端通信时，除登录请求外，每个请求头中都需要携带一个 JWT 令牌，以鉴定权限。
